HTML Essentials: File Path, Boilerplate, Structure Tags

File Path in HTML

  • Absolute Path: Full path from the root directory.
    /images/photo.jpg
  • Relative Path: Path relative to the current file.
    images/photo.jpg
  • Parent Directory: Use ../ to go up one level.
    ../assets/logo.png

HTML Boilerplate Code

The basic structure every HTML file should start with:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>
  
</body>
</html>

HTML Structure Tags: <div>, Class, ID, Header, Footer, Main

  • <div>: Generic container for grouping content.
  • class: Attribute for styling multiple elements together.
  • id: Attribute for uniquely identifying a single element.
  • <header>: Defines the top section (logo, nav, etc.).
  • <footer>: Defines the bottom section (copyright, links).
  • <main>: Main content of the page.
Tag/Attribute Purpose Example
<div> Generic container <div>Content</div>
class Group for styling <div class="box">...</div>
id Unique identifier <div id="header">...</div>
<header> Top section <header>Site Title</header>
<footer> Bottom section <footer>Copyright 2025</footer>
<main> Main content <main>Article</main>

Example: Page Structure

<header>
  <h1>My Website</h1>
</header>
<main>
  <div class="content">
    <p>Welcome to my site!</p>
  </div>
</main>
<footer id="site-footer">
  © 2025 My Website
</footer>
