HTML Forms 📝

Forms are one of the most important parts of web development. Whenever you log in, sign up, or submit details on a website, you are using an HTML form.
Let's break it down in simple terms.

1️⃣ Label & Input 🏷️

  • label → Describes what the field is about.
  • input → The actual field where the user types something. 
<label>Name :</label>
    <input type="text">

2️⃣ Connecting Label and Input (for - id) 🔗

  • Use for in <label> and id in <input> to connect them.
  • Clicking the label focuses the input box.
  • Makes forms more accessible (helpful for screen readers).
<label for="email">Email :</label>
    <input type="email" id="email">

3️⃣ Name–Value Pair (What Goes to the Backend) 📨

  • When you fill a form and hit submit, your browser sends data to the backend as name=value pairs.
  • Radio buttons example:
<label for="male">Male</label>
    <input type="radio" id="male" name="gender" value="male">
    <label for="female">Female</label>
    <input type="radio" id="female" name="gender" value="female">
    <label for="other">Other</label>
    <input type="radio" id="other" name="gender" value="other">

✅ In short:

  • label + input → Basic building block.
  • for + id → Connects label to input for better UX.
  • name=value → Data sent to backend.
